# Zero Effort Planner (ZEP)

Zero Effort Planner is a desktop productivity application developed in C# using Windows Forms.

The application helps users organize their daily tasks, manage monthly budgets, and create shopping lists.

## Features

- To-Do List for managing daily tasks
- Budget Calendar for weekly budget planning
- Shopping List for organizing purchases

## Technologies Used

- C#
- .NET Framework
- Windows Forms
- Visual Studio

## Project Structure

The application contains three main modules:

1. To-Do List – manage daily activities
2. Budget Calendar – calculate weekly budget from a monthly amount
3. Shopping List – create and manage product lists
